# JavaScript Mini Projects 

This repository contains multiple beginner-friendly mini projects built using **HTML, CSS, and JavaScript**.  
These projects helped me understand core JavaScript concepts and improve my frontend development skills.


# Projects Included

#1,Calculator
- Perform basic arithmetic operations (+, −, ×, ÷)
- Clean and simple UI
- Built using JavaScript DOM manipulation

#2, Background Color Changer
- Changes background color dynamically on button click
- Helps understand event handling and DOM updates

#3, Digital Clock
- Displays real-time digital clock
- Uses JavaScript Date object and timing functions

#4, QR Code Generator
- Generates QR codes based on user input
- Simple and interactive UI


>> Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
